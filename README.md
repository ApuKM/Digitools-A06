DigiTools - AI Tools Marketplace

Description:
DigiTools is a modern web application where users can explore, purchase, and manage AI-powered tools. It provides a clean UI and smooth user experience for browsing tools, adding them to cart, and proceeding to checkout.

Technologies Used:
  1.React.js
  2.Tailwind CSS
  3.JavaScript (ES6+)
  4.Daisy ui
  5.React toastify

Features:
Smart Cart System:
Users can add AI tools to the cart, view selected items, and manage them easily with real-time updates.
Checkout Flow:
Simple and intuitive checkout process with coin deduction logic.
Responsive Design:
Fully responsive UI that works smoothly across mobile, tablet, and desktop devices.
